# CoffeeMaker - программа по управлению кофеваркой через браузер.

## Возможности:
1) Включение/выключение кофеварки.
2) Проверка наличия зерен/молока/воды в кофеварки.
3) Функция наполнения зерен/молока/воды в кофеварку.
4) Приготовление различных видов кофе( латте, капучино, американо, эспрессо, черный кофе) . 
5) В зависимости от типа и крепости кофе тратится разное количество зерен/молока/воды.
6) Все манипуляции с кофеваркой записываются в БД Postgres.
7) Взаимодействие с интерфейсом кофеварки реализуется через SwaggerUI. 
Ссылка: http://localhost:8080/swagger-ui/index.html?configUrl=/api-docs/swagger-config#/
8) Перед тем, как сварить кофе, необходимо включить кофеварку.

## Используемый стек технологий:
Spring Boot, PostgresSQL, SwaggerUI
